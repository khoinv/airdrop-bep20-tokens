# Airdrop BEP-20 Tokens

## Overview

- Airdrop tokens using [Disperse.app](https://bscscan.com/address/0xd152f549545093347a162dce210e7293f1452150)

## Prerequisites

The following prerequisites are required to be installed on your system:

- NodeJS 12.13.0
- Yarn (optional)

## Execution

```sh
node index.js
```
